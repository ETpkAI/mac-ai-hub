## What Makes Apple Silicon Special?

Apple's M1/M2/M3 chips feature **Unified Memory Architecture (UMA)**, which means the same high-bandwidth memory is shared between CPU and GPU. This is a game-changer for AI inference:

:::tip Key Advantage
Unlike traditional PCs where you need expensive dedicated GPUs with separate VRAM, your Mac's unified memory can be fully utilized for running large language models.
:::
