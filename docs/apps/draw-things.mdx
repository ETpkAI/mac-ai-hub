# Draw Things: The Best Native App for AI Image Generation

**Draw Things** is hands-down the best way to generate AI images on your Mac. It's a free, native macOS app that runs entirely on your device using Apple's Metal API.

## Installation

### Method 1: App Store (Recommended)

1. Open the **App Store** on your Mac
2. Search for "Draw Things"
3. Click **Get** to download

### Method 2: Direct Download

Visit [drawthings.ai](https://drawthings.ai) to download directly.

## First-Time Setup

### 1. Download a Model

When you first open Draw Things, you'll need to download a model:

1. Go to **Settings** (gear icon) ‚Üí **Models**
2. Recommended starter model: **Stable Diffusion XL Base**
3. Click **Download** and wait for it to complete

:::warning Storage Space
SDXL models are ~6GB. Ensure you have sufficient disk space before downloading.
:::

### 2. Generate Your First Image

1. Type a prompt in the text box, e.g.:
   ```
   A majestic snow leopard on a mountain peak, 
   golden hour lighting, photorealistic, 8K
   ```

2. Click **Generate**

3. Wait 30-60 seconds (first generation is slower due to model loading)

## Troubleshooting

### "Out of memory" errors
- Reduce image size
- Close other applications
- Enable "Low Memory Mode" in settings

### Slow generation
- First generation loads the model (normal to be slow)
- Check Activity Monitor for memory pressure
- Restart the app to clear cache

### Model not loading
- Verify download completed
- Check **~/Library/Application Support/Draw Things/** for corrupted files
