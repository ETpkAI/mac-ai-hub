# Draw Things：最佳原生 AI 图像生成应用

**Draw Things** 绝对是在 Mac 上生成 AI 图像的最佳方式。它是一款免费的原生 macOS 应用，完全在您的设备上使用 Apple 的 Metal API 运行。

## 为什么选择 Draw Things？

| 特点 | 优势 |
|------|------|
| 🍎 **原生 Mac 应用** | 专为 macOS 构建，不是网页封装 |
| 🔒 **100% 本地运行** | 无需互联网，完全隐私 |
| ⚡ **Metal 优化** | 使用 Apple GPU 加速实现快速生成 |
| 🆓 **完全免费** | 无订阅，无 API 费用 |
| 📱 **iPad 支持** | 同一应用可在搭载 M 系列芯片的 iPad 上运行 |

:::tip 无需命令行！
与许多 AI 工具不同，Draw Things 拥有精美的图形界面。非常适合想要专注于艺术创作而非终端命令的创作者。
:::

## 安装

### 方法 1：App Store（推荐）

1. 在 Mac 上打开 **App Store**
2. 搜索 "Draw Things"
3. 点击 **获取** 下载

### 方法 2：直接下载

访问 [drawthings.ai](https://drawthings.ai) 直接下载。

## 首次设置

### 1. 下载模型

首次打开 Draw Things 时，您需要下载一个模型：

1. 进入 **设置**（齿轮图标）→ **模型**
2. 推荐入门模型：**Stable Diffusion XL Base**
3. 点击 **下载** 并等待完成

:::warning 存储空间
SDXL 模型约 6GB。下载前请确保有足够的磁盘空间。
:::

### 2. 生成您的第一张图片

1. 在文本框中输入提示词，例如：
   ```
   雪山之巅的雪豹，金色夕阳光线，
   逼真写实，8K 高清
   ```

2. 点击 **生成**

3. 等待 30-60 秒（首次生成较慢，因为需要加载模型）

## Apple Silicon 推荐设置

### 8GB Mac（M1/M2/M3 基础版）

| 设置 | 值 |
|------|-----|
| 模型 | SD 1.5 或 SDXL（启用内存优化） |
| 图片尺寸 | 512×512 或 768×768 |
| 步数 | 20-25 |
| 引导系数 | 7.0 |

### 16GB+ Mac（Pro/Max）

| 设置 | 值 |
|------|-----|
| 模型 | SDXL、SD 3 或自定义微调模型 |
| 图片尺寸 | 1024×1024 或更高 |
| 步数 | 25-30 |
| 引导系数 | 7.5 |

## 获得更好效果的技巧

### 提示词技巧

```
✅ 好的提示词："赛博朋克武士肖像，霓虹城市背景，
         雨中倒影，电影级光效，高度细节"

❌ 差的提示词："城市里的武士"
```

### 负面提示词
添加这些以避免常见问题：
```
模糊, 低质量, 变形, 丑陋, 
手部变形, 多余手指, 水印
```

### 种子一致性
锁定种子数以在提示词变化时重现相似结果。

## 故障排除

### "内存不足"错误
- 减小图片尺寸
- 关闭其他应用程序
- 在设置中启用"低内存模式"

### 生成缓慢
- 首次生成需要加载模型（正常较慢）
- 在活动监视器中检查内存压力
- 重启应用清除缓存

### 模型无法加载
- 验证下载是否完成
- 检查 **~/Library/Application Support/Draw Things/** 是否有损坏文件
